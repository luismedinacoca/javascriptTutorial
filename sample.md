<!-- Heading -->
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

<!-- Italics -->
## italics:
*This text* is italics
_this text_ is italics

<!-- Strong -->
## Strong:
**This text** is strong
__this text__ is strong

<!-- Strikethrough -->
## Strikethrough:
~~This text~~ is strikethrough

<!-- Horizontal Rule -->
## Horizontal Rule:
---
___


<!-- Blockquote -->
## Blockquote:
> this is a quote


<!-- Link -->
## Link:
[Google](https://www.google.com "Google page")

[readme.so](https://readme.so/es/editor "editing readme file")


<!-- UL -->
## Unordered list:
* Item 1
* Item 2
	* nested item 1 (tab)
	* nested item 2 (tab) 

<!-- OL -->
## Ordered List:
1. Item 1
    1. nested item 1
    1. nested item 2
    1. nested item 3
1. Item 2
1. Item 3

## Code blocks:

```bash

```

### javascript code:
```javascript
function add(num01, num02){
    return num01 + num02;
}
```

```ruby
def add(num01, num02){
    num01 + num02;
}
```
